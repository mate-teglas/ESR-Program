ESR CLEAN REVIEW / EDIT BROWSER
================================

REVIEW MODE (default)
- Large data/fit plot and residual panel.
- The individual fitted domain curves remain visible by default.
- Toggle Components to hide/show the component curves at any time.
- Toggle Domain labels independently.
- Previous/Next spectra remain available as muted context.
- A/Q/X: accept / needs review / exclude.
- [ and ]: previous/next spectrum needing review.
- E: enter Edit mode.

EDIT MODE
- Click a fitted resonance or its D1-D4 label.
- Press 1,2,3,4 to rename it.
- If the new ID already exists at that spectrum, the two IDs are swapped.
- L toggles the lock on the selected assignment.
- Rename scope can be current / forward / backward / all.
- Lock assignment prevents automatic tracking from overwriting manual changes.
- T sets the current four-component spectrum as the tracking reference.
- Automatic and manual-center refitting controls are shown only in Edit mode.
- M toggles center-picking mode; Enter refits from selected centers.

The Components checkbox is intentionally enabled by default so the final total
fit can always be inspected as the sum of the individual domain fits.
